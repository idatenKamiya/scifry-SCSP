ProtocolIR final summary

Current state:
- OpenRouter strict structured-output parser is wired through OPENROUTER_API_KEY.
- The main parser stops the run when OpenRouter, the key, or schema support is invalid.
- Bayesian IRL training writes full posterior samples, MAP estimate, posterior means, 95% credible intervals, R-hat, ESS, and learned weights.
- The compiler path is typed IR -> verifier -> reward score -> repair -> Opentrons code -> real simulator -> audit.
- Architecture is documented in ARCHITECTURE.md.

Run:
cd ProtocolIR
.\.venv\Scripts\Activate.ps1
python test_installation.py
$env:OPENROUTER_API_KEY="your_openrouter_key_here"
$env:PROTOCOLIR_MODEL="openrouter/free"
python check_openrouter.py
python train_reward_model.py
python main.py --stress-demo -o stress_output

Security:
- Keep real API keys in environment variables only.
