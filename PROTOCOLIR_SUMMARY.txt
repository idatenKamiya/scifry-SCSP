ir = pir.build_ir(grounded)
violations = pir.verify_ir(ir)
ir_fixed, repairs = pir.repair_ir(ir, violations)
script = pir.compile_to_opentrons(ir_fixed)
result = pir.simulate_opentrons_script(script)
print(f"Violations fixed: {len(repairs)}")

================================================================================
THE 9-LAYER ARCHITECTURE
================================================================================

Layer 1: Parser
  Input: Raw natural language protocol
  Output: Semantic actions (structured JSON)
  Tool: Claude LLM for extraction

Layer 2: Grounder
  Input: Semantic actions with abstract locations
  Output: Actions mapped to concrete deck positions
  Tool: Heuristic location resolver

Layer 3: IR Builder
  Input: Grounded actions
  Output: Typed machine-readable IR (IROp operations)
  Tool: Deterministic IR compiler

Layer 4: Verifier
  Input: IR operations
  Output: List of safety violations (if any)
  Tool: Hard constraint checker

Layer 5: Reward Scorer
  Input: IR + violations
  Output: Numerical safety score
  Tool: Learned reward function from demonstrations

Layer 6: Repair Policy
  Input: IR + violations
  Output: Repaired IR + list of fixes applied
  Tool: Deterministic repair rules

Layer 7: Compiler
  Input: Verified IR
  Output: Opentrons Python script
  Tool: Deterministic code generator

Layer 8: Simulator
  Input: Generated Opentrons script
  Output: Simulation result (pass/fail)
  Tool: Opentrons built-in simulator

Layer 9: Audit
  Input: Entire pipeline history
  Output: Human-readable safety report
  Tool: Report generator
