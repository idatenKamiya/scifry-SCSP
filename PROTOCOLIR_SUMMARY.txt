ProtocolIR summary

ProtocolIR is a strict, reward-guided compiler for safe lab automation.

Pipeline:
1. Parse protocol text to structured JSON with OpenRouter strict structured output.
2. Ground semantic actions to OT-2 deck locations and wells.
3. Build typed robot IR.
4. Verify hard physical constraints.
5. Score reward features with the Bayesian IRL posterior mean.
6. Repair unsafe behavior deterministically.
7. Compile verified IR to Opentrons Python.
8. Validate with the real Opentrons simulator.
9. Generate audit artifacts.

Required run order:
cd ProtocolIR
.\.venv\Scripts\Activate.ps1
python test_installation.py
$env:OPENROUTER_API_KEY="your_openrouter_key_here"
$env:PROTOCOLIR_MODEL="openrouter/free"
python check_openrouter.py
python train_reward_model.py
python main.py --stress-demo -o stress_output
